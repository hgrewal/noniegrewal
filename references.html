Websites and References

Here you can find a list of websites and other references displaying my interests in
software engineering as well as my passion for Computing and Technology.

This is a great beginner's tutorial for

Bootstrap is a great tool for web develop-
ment. It provides tutorials for CSS and
and Javascript. It also provides source

Stack Overflow is a great resource for all
programmers. It is question answer
forum where coders can go to ask one

Women in Computing, or WIC@UCSD is a
student organization that strives to recruit
and retain more women into Computer Science.

Grace Hopper Celebration of Women in
Computing is a yearly conference and
gathering of Women Technologists.

Hour of Code was a movement started at the
end of 2013 that strives to teach coding
to as many people as possible.
